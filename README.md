## Requirements

The list of tools required to build and run the project:

* Open JDK 11.x+
* Apache Maven 3.5.x+

## Building

In order to build project use:

```bash
mvn clean package
```

## Running

In order to run using Open Liberty Application server use:

```bash
mvn -P liberty liberty:dev
```
